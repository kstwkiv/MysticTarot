# Assets

Place tarot card images here under:
- `cards/major/` — Major Arcana card images (fool.jpg, magician.jpg, etc.)
- `cards/minor/` — Minor Arcana card images (ace-of-wands.jpg, etc.)

Images are referenced by the TarotDeckService with paths like `/assets/cards/major/fool.jpg`.
The app works without images — cards display symbolic icons as placeholders.
